[Amarok 1.4](http://amarok.kde.org/) has been released and all I can say is WWWWOOOOOOOWWWW. This release is amazing, guys. I've been waiting for it to show up in my kubuntu updates list and it hasn't, so tonight I hunted a bit and found [this announcement](http://kubuntu.org/announcements/amarok-1.4.php) showing how to get it for [dapper](http://www.ubuntu.com/). As I type this, I'm transferring 1.4G to my iPod nano, and it looks like it's going to also transfer the album cover artwork along with it. Kick butt!!!

For those who have been living in a hole for the last few years, amarok is an absolutely amazing and free music manager that uses the Qt toolkit, and is thusly available for Linux, *BSD, OS X, and probably Windows too, though I don't honestly care enough to look. I definitely heart this new release--the best one yet!! =:)
