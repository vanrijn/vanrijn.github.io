For whatever reason, I cannot get my fonts in konsole (SUSE 10.2) to not look like crap. Before I grew attached to KDE and konsole, I used blackbox and Eterm (for like 5+years???). And Eterm did a fantastic job of using the classic (and really good) artwiz fonts like smoothansi, fkp, snap, mints, edges, anorexia. Konsole, apparently, cannot use the fonts that I'd used (and kept) for years with Eterm. So I stumbled upon this sourceforge project which has resurrected the glorious artwiz fonts and allows them to be used in KDE/GTK applications once more (including konsole).

All hail Artwiz!! (where did you go, Artwiz!?!) All hail Oliwier Ptak!!!
